# AI Usage Disclosure

## Tools Used

- **ChatGPT** (GPT-5.5) – used as an assistant throughout the project, both for general directions and coding assistance

## How It Was Used

- **Chat based code and structure suggestions**: The AI was used to assist in defining the project requirements, helping with setting up the required structure and generating very basic code drafts,
- which was then reviewed and completed by myself. 

- **Fully generated code**: There are a few files and segments that were generated by the AI entirely, mentioned below and marked in the source code with `// AI-generated`  `// end AI-generated`.
- This has been done in certain areas where the code was basic enough that no relevant changes could be made, or in cases in which I did not succeed in writing the code myself or updating
- the basic drafts given by the AI.

- **Bug-fixing**: In the earlier stages of development, I attempted to use assets for the graphics of the game, but I did not succeed. The current render used in the application came about
- at the suggestion of the AI assistant.

## Fully AI-Generated Files / Regions

- `Map.cs`: Both the concept of using a string array and the code itself have been fully AI generated
- `Renderer code`: Most of the render handling has also been fully AI generated, in the  "Game.cs" file. It has been marked as such.
